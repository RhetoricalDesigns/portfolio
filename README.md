# Rhetorical Designs — Portfolio Editions

A multi-style portfolio for **Duane Steffens** (Rhetorical Designs, Hermanus, South Africa — remote).
One content set, ten distinct visual editions. Built from the live site
[rhetoricaldesigns.website](https://rhetoricaldesigns.website).

## Editions

| # | Style | Lane |

| 01 | Editorial Brutalist | Bold / in-your-face |
| 02 | Folio & Craft | Elegant / gallery |
| 03 | Tech-Noir Coastal | Dark / neon |
| 04 | Agency Partner | Studio pitch deck |
| 05 | Swiss Precision | Corporate / exact |
| 06 | Hermanus Coastal | Warm / natural |

| 08 | Kinetic Playground | Motion / interactive |
| 09 | CLI Resume | Terminal / dev |

## Run locally

```bash
cd portfolio
python3 -m http.server 8099
# open http://localhost:8099/
```

`index.html` is the gateway — it links to all ten editions.

## Deploy

Static site — drop the folder on GitHub Pages, Netlify, Cloudflare Pages, or any static host.
No build step required.
